# emoji-match 🐸

Browser-based memory matching game with emoji themes, multiple difficulty
levels, timers, best scores, and support for future custom icon sets.

## How to Play

Open `index.html` in any modern browser — no build step, no dependencies.

- Pick a **theme** (Emoji, Animals, Fruit, Faces)
- Pick a **level** (2×2 → 4×4 → 6×6 → 8×8)
- Flip cards to find matching pairs
- Beat your best time and fewest picks

Completing a level unlocks the next one.
